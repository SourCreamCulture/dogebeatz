## Contributing

Thanks for considering to contribute! here's how to set up your environment:

1. fork this repo then clone it
2. copy `config.example.js` to `config.js`. in linux or mac: `cp config.example.js config.js`;
3. [get your access & refresh tokens](https://moonstone.folf.party/#/main/main/general/tokens) then put them in `config.js`
4. run `node createbot.js` and save token you get, then put it in `config.js` under `botToken`;
5. run `index.js`!
